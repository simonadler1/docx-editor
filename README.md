# DOCX Editor

A browser-based WYSIWYG DOCX editor with template variable support.

<!-- Add video here -->

## Quick Start

```bash
bun install
bun run dev
```

## Features

- Open and edit DOCX files in the browser
- WYSIWYG editing with Microsoft Word fidelity
- Template variable insertion (`{{variable}}`)
- Export back to DOCX
